=head1 NAME

headpin - Headpin command line tool

=head1 SYNOPSIS

  headpin [options] command [parameter1] [parameterN] ...

    Options include:
      [ -h | --help ]
      [ -u | --username ]
      [ -p | --password ]
      [ --host ]

Commands and parameters are described below.

=head1 DESCRIPTION

The B<headpin> program is an command line application for interaction with a
Headpin instance. It can be used by Headpin administrators and users to
execute regular tasks. There is a built-in shell avaiable that can be used
for scripting purposes.
