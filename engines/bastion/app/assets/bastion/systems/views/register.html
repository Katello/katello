System Registration

To register a system to this server, run the following commands within a console on the client system.

Install the pre-built bootstrap RPM. This will configure the system to use this server.

rpm -Uvh {{ baseURL }}/pub/candlepin-cert-consumer-latest.noarch.rpm

Use Subscription Manager to register.

subscription-manager register --org="{{ organization }}"

Consuming Content From a Node
{{ "Selected Node:" | translate }} 
subscription-manager register --org="{{ organization }}" --baseurl="https://{{  selectedNode.hostname }}/pulp/repos"
